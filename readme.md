# 🐾 PetShop API

A RESTful API for managing pet shop operations, including appointments, customers, pets, products, services, and staff.

## 🚀 Technologies

- Node.js
- Fastify
- Prisma ORM
- PostgreSQL
- TypeScript
- Nodemailer




## 📦 Installation

```bash
git clone https://github.com/leonardo849/petshop.git
cd petshop
npm install
npm run generate
```

## .Env
- DATABASE_URL: string
- PORT: string
- SALTS: string
- SECRET: string
- USER: string
- PASSWORDEMAIL: string
- HOSTEMAIL: string
- PORTEMAIL: string


## Run
```bash
npm run build
npm run start
```
